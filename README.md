# 🌐 Personal Portfolio Website

A modern, responsive Personal Portfolio Website built using HTML, CSS, and JavaScript to showcase my skills, projects, and contact information.

---

## 👩‍💻 About The Project

This portfolio website presents:

- 👋 Introduction  
- 🙋 About Me section  
- 🛠 Technical & Professional Skills  
- 💻 Tools & Technologies  
- 📂 Projects  
- 📞 Contact Information  

It is fully responsive and works on desktop, tablet, and mobile devices.

---

## 🚀 Features

- Responsive Design (Mobile Friendly)
- Fixed Navigation Bar
- Modern Gradient UI
- Project Cards with Hover Effects
- Download CV Button (JavaScript)
- Contact Section with Clickable Email & Phone
- Clean and Organized Code Structure

---

## 🛠 Built With

- HTML5

- JavaScript

- Bootstrap Icons
